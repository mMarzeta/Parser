Dane są pobierane z folderu src/samples/
Muszą być zapisane w formacie .txt

Aby uruchomić program w terminalu należy:
javac Parser.java
java Parser

Lub jeśli jest to możliwe po prostu wcisnąć dwukrotnie ikony pliku
Parser.class
